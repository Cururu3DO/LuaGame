# LuaGame 0.0.1

LuaGame is a simple 2D game library for Lua focused on simplicity and beginner-friendly development.

## Features

- Window creation
- Rendering
- Rectangles
- Image rendering
- Text rendering
- Keyboard input

## Requirements

- Lua 5.4
- SDL2
- SDL2_image
- SDL2_ttf
- GCC

## Installation

Ubuntu/Debian:

```bash
sudo apt install build-essential lua5.4 liblua5.4-dev \
libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev pkg-config
